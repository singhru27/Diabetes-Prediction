Project Overview:
Conducted an investigation into the efficacy of various ML algorithms in predicting readmission rates and length of stays for diabetic hospital patients. 

Results are presented in "Analysis Results". Raw graphs, data, and python scripts used in the analysis of the data are included in "Graphs", "Data", and "Python Scripts" respectively. Unzip the files in the "Data" files and update the paths to the datafiles for the scripts to run properly
